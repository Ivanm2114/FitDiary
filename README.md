# Fitdiary

Fitdiary — мобильный дневник питания и рецептов для тех, кто хочет контролировать калории и питаться осознанно. Приложение помогает фиксировать приемы пищи, видеть прогресс по калориям и вдохновляться полезными рецептами.

## Что умеет
- Дневник питания с выбором даты и списком приемов пищи.
- Быстрое добавление блюд с калориями и редактирование.
- Отображение нормы калорий и прогресса за день.
- Каталог полезных рецептов с изображениями и подробным описанием.
- Профиль пользователя с целями и персональными параметрами.

## Зачем нужно
Fitdiary помогает держать питание под контролем: вы видите, сколько калорий уже съедено, какие блюда были в течение дня, и получаете идеи для здоровых рецептов. Это простой инструмент для тех, кто хочет похудеть, поддерживать форму или набрать массу.

## Техническая часть (кратко)
Приложение состоит из экранов (UI), логики (ViewModel), слоя данных (API и локальное хранение) и сервисных компонентов (метрики).

Основные технологии:
- Android + Kotlin + Jetpack Compose (интерфейс)
- Navigation Compose (навигация)
- Retrofit + OkHttp + Kotlinx Serialization (сетевые запросы)
- DataStore (локальное хранение профиля)
- Coroutines/Flow (асинхронная работа)
- Coil (загрузка изображений)

Backend:
- Flask + SQLite
- Деплой через Docker
- Сервер предоставляет API для регистрации/логина, рецептов, дневника питания, метрик и рейтингов.
- В проекте используется адрес: `http://192.168.10.14:5000/`
